# Raul Palma Portfolio (React 2026)

Portafolio migrado desde HTML/CSS/JS legacy a stack moderno:

- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion
- Lucide Icons

## Scripts

- `npm run dev`: entorno local
- `npm run build`: build produccion
- `npm run preview`: vista previa del build
- `npm run deploy`: publica `dist/` en GitHub Pages (rama `gh-pages`)

## Deploy en GitHub Pages

1. Asegura que el repositorio remoto sea `Rpalmak.github.io` (sitio de usuario).
2. Ejecuta:
   - `npm install`
   - `npm run deploy`
3. En GitHub, verifica Pages apuntando a la rama `gh-pages`.

## Assets heredados

Las carpetas legacy originales se conservaron, y los assets usados por React estan en:

- `public/img`
- `public/data`
